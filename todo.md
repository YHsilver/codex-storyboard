优化生成逻辑，调用 api 逻辑，参数逻辑。
1. 生成时，只携带当前分镜的前序已选择的素材文件，不要携带其他分镜的内容
2. prompt优化（Skill）：prompt 需要遵循模型配置，其中 prompt-prefix 是固定添加的，prompt 模板是最终发送的提示词模板，需要根据剧情修改内容。在prompt中需要使用[图1]之类的方式来引用对应的图片
3. 文件命名规范: 所有生成的文件都需要清晰的命名，codex不需要读取文件内容，只需要根据文件名索引和判断是否存在素材即可，减少 codex 的 token消耗
    - 命名格式：物料 pxxx-cx-m1 (项目-分镜x-物料1)、故事板: pxx-cx-s1 (项目-分镜x-故事板1)、视频 pxx-cx-v1(项目-分镜x-视频1)
